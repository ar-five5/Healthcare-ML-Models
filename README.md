# Healthcare ML Models

A collection of ML projects built during my 6-month AI/ML training program at i-Hub, IIIT Hyderabad (Sep 2024 - Mar 2025).

These were some of my first ML projects, focused on applying classification and regression techniques to healthcare datasets.

## Projects

**Diabetes Prediction** — Classification model to predict diabetes risk using patient clinical data. Built with Logistic Regression, Decision Trees, and SVM. Includes feature selection and cross-validation.

**Heart Disease Prediction** — Classification pipeline to predict heart disease likelihood from patient records. Uses Logistic Regression, Decision Trees, and SVM with preprocessing for missing values and class imbalance.

**COVID-19 Regression Analysis** — Regression analysis on a COVID-19 dataset to model case growth trends and outcomes. Covers data cleaning, feature engineering, and visualization of epidemiological patterns.

## Tools

Python, Scikit-learn, Pandas, NumPy, Seaborn, Matplotlib

## Note

These were built as course projects while I was starting out with ML. They're not production-grade, but they gave me a foundation in working with health data, thinking about prediction problems in a clinical context, and interpreting model outputs beyond just accuracy scores.
